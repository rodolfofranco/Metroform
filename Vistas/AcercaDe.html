Acerca De
MetroForm es una aplicación web desarrollada por estudiantes de la materia Sistemas de Información de la UNIMET. Consiste en una plataforma para la automatización de evaluaciones del desempeño de los profesores de la universidad.
Términos y Condiciones
Al registrarte en esta página y al hacer uso de MetroForm estás accediendo a los términos de la empresa. En donde podemos usar la información pública, no tu data sensible, para poder auditar y gestionar toda la información que obtenemos para poder realizar mejoras en nuestro sistema y la base de datos.
Asociaciones
Esta aplicación fue desarrollada en asociación con el CETIC y autoridades de la Universidad Metropolitana.
Especificaciones
MetroForm es un proyecto diseñado para las funcionas básicas del desempeño de los docentes de la UNIMET. En esta aplicación podrás acceder a tu información académica, inscribirte en materias para realizar evaluaciones a tus profesores y para los profesores, estos podrán realizar sus autoevaluaciones y ver su desempeño generalizado.
Herramientas
Se utilizó para el desarrollo colaborativo web la plataforma Cloud9 --> c9.io
Recursos
Se utilizó Node.js que es un entorno en tiempo de ejecución multiplataforma, de código abierto, para la capa del servidor basado en el lenguaje de programación ECMAScript, asíncrono, con I/O de datos en una arquitectura orientada a eventos y basado en el motor V8 de Google. --> Node.js

Además se utilizó Sequelize que es un ORM que está diseñado para trabajar con Node.JS e IO.JS. Está escrito en JavaScript puro y soporta muchos motores de base de datos como MySQL, Postgres, SQLite y MSSQL. --> Sequelizejs.com

Finalmente se utilizó MySQL que es un sistema de gestión de bases de datos relacional desarrollado bajo licencia dual GPL/Licencia comercial por Oracle Corporation y está considerada como la base datos open source más popular del mundo, y una de las más populares en general junto a Oracle y Microsoft SQL Server, sobre todo para entornos de desarrollo web. --> MySQL.com
